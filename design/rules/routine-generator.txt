If user-input has a frequency of 3 AND workout-split is not assigned
THEN
    workout-split is full-body
    focus on FULL-BODY

If user-input has a frequency of 4 AND workout-split is not assigned
THEN
    workout-split is upper-lower
    focus on UPPER-LOWER

If user-input has a frequency of 6 AND workout-split is not assigned
THEN
    workout-split is push-pull-leg
    focus on PUSH-PULL-LEG


If there is a workout-split assigned with name
THEN
    if workout-split is full-body then focus on FULL-BODY
    if workout-split is upper-lower then focus on UPPER-LOWER
    if workout-split is push-pull-leg then focus on PUSH-PULL-LEG


; Reps determination
For each exercise-slot with min-reps AND max-reps
    If user-input has goal strength AND exercise-slot has min-reps not 1 or max-reps not 5
    THEN
        modify exercise-slot to have min-reps 1 AND max-reps 5

    If user-input has goal hypertrophy AND exercise-slot has min-reps not 8 or max-reps not 12
    THEN
        modify exercise-slot to have min-reps 8 AND max-reps 12

    If user-input has goal endurance AND exercise-slot has min-reps not 15 or max-reps not 100
    THEN
        modify exercise-slot to have min-reps 15 AND max-reps 100

; Set assignment
For each exercise-slot with primary-muscle-group AND sets 0
    If muscle-group with name of primary-muscle-group has high priority (priority 1)
    THEN
        modify exercise-slot to have sets 3

    If muscle-group with name of primary-muscle-group does not have high priority (priority 2,3,4)
    THEN
        modify exercise-slot to have sets 2

; Determine global order
For each exercise-slot with primary-muscle-group
    modify exercise-slot to have global-order equal to (day-order * 100 + exercise-order)

; Exercise selection
; Sub-muscle-group assignment for first exercise
IF there is a muscle-group with name main-mg, 
    AND there is a sub-muscle-group with name sub-mg AND sub-mg is a sub-muscle-group of main-mg AND sub-mg has high priority,
    AND there is a exercise-slot s have global-order g AND s has primary-muscle-group with same name as main-mg AND s does not have targeted-sub-muscle-group assigned,
    AND there is not a exercise-slot s2 with global-order less than g AND
    s2 has primary-muscle-group with same name as main-mg
THEN 
    modify exercise-slot s to have targeted-sub-muscle-group with same name as sub-mg

; Sub-muscle-group assignment for next exercise
IF there is an exercise-slot with global-order g-prev AND primary-muscle-group main-mg AND targeted-sub-muscle-group prev-smg already assigned,
    AND there is a sub-muscle-group prev-smg belonging to main-mg with priority prev-p, 
    AND there is a sub-muscle-group sub-mg belonging to main-mg with priority exactly one greater than prev-p, 
    AND there is an exercise-slot s with global-order g-order greater than g-prev AND primary-muscle-group main-mg AND no targeted-sub-muscle-group assigned, 
    AND there is no other exercise-slot with primary-muscle-group main-mg AND no targeted-sub-muscle-group assigned whose global-order falls between g-prev AND g-order
THEN
    modify exercise-slot s to have targeted-sub-muscle-group of sub-mg

; Sub-muscle-group loopback
IF there is an exercise-slot with global-order g-prev AND primary-muscle-group main-mg AND targeted-sub-muscle-group prev-smg already assigned, 
    AND there is a sub-muscle-group prev-smg belonging to main-mg with priority prev-p, 
    AND there is no sub-muscle-group belonging to main-mg with priority one greater than prev-p, 
    AND there is a sub-muscle-group sub-mg belonging to main-mg with high priority, 
    AND there is an exercise-slot s with global-order g-order greater than g-prev AND primary-muscle-group main-mg AND no targeted-sub-muscle-group assigned, 
    AND there is no other exercise-slot with primary-muscle-group main-mg AND no targeted-sub-muscle-group assigned whose global-order falls between g-prev AND g-order
THEN
    modify exercise-slot s to have targeted-sub-muscle-group of sub-mg

; Swap sub-muscle-group based on day order
IF there is an exercise-slot s1 with day-order day-order, with primary-muscle-group main-mg, with targeted-sub-muscle-group sub-mg1, 
    AND there is an exercise-slot s2 with the same day-order AND same primary-muscle-group main-mg AND targeted-sub-muscle-group sub-mg2, 
    AND sub-mg1 has priority priority1,
    AND sub-mg2 has priority priority2, 
    AND priority1 is greater than priority2, 
    AND the fact index of s1 is less than the fact index of s2
THEN
    
    modify exercise-slot s1 to have targeted-sub-muscle-group of sub-mg2,
    AND modify exercise-slot s2 to have targeted-sub-muscle-group of sub-mg1

; Exercise selection

; User prefer compound
; Salience 3
IF there is an exercise-slot s with a targeted-sub-muscle-group sub-mg already assigned AND exercise s-exercise not assigned, 
    AND there is an exercise with id exercise-id that targets sub-mg AND equipment eq AND is a compound movement, 
    AND there is a user-input with a type matches eq, 
    AND there is no other exercise-slot has already been assigned exercise-id
THEN
    assign exercise-id as the exercise of exercise-slot s

; User prefer isolation
; Salience 2
IF there is an exercise-slot s with a targeted-sub-muscle-group sub-mg already assigned AND exercise s-exercise not assigned, 
    AND there is an exercise with id exercise-id that targets sub-mg AND requires equipment equipment AND is a isolation movement, 
    AND the user's preferred exercise type matches equipment, 
    AND there is no other exercise-slot has already been assigned exercise-id
THEN
    assign exercise-id as the exercise of exercise-slot s

; Asign exercise without user preference, compound
; Salience 1
IF there is an exercise-slot s with a targeted-sub-muscle-group sub-mg already assigned AND exercise s-exercise not assigned, 
    AND there is an exercise with id exercise-id that targets sub-mg AND is a compound movement, 
    AND there isno other exercise-slot has already been assigned exercise-id
THEN
    assign exercise-id as the exercise of exercise-slot s

; Asign exercise without user preference, isolation
; Salience 0
IF there is an exercise-slot s with a targeted-sub-muscle-group sub-mg already assigned AND exercise s-exercise not assigned, 
    AND there is an exercise with id exercise-id that targets sub-mg AND is a isolation movement, 
    AND there is no other exercise-slot has already been assigned exercise-id
THEN
    assign exercise-id as the exercise of exercise-slot s


; Asign exercise without user preference, fallback, either isolation or compound
; Salience -1
IF there is an exercise-slot s with a targeted-sub-muscle-group sub-mg already assigned AND exercise s-exercise not assigned, 
    AND there is an exercise with id exercise-id that targets sub-mg, 
    AND there is no other exercise-slot has already been assigned exercise-id
THEN
    assign exercise-id as the exercise of exercise-slot s
