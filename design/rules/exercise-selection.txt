IF all exercise slots are assigned an order and priority and muscle group THEN
    FOR EACH days
        FOR EACH exercise-slots sorted by order
            Assign an exercise to the slot # See below

 # Exercise selection logic

FOR ALL exercise-slot s:

IF exercise ex is assigned to exercise-slot s THEN
    ex is not assigned to any other exercises-slot of the same day.

# Filter exercise based on priority
IF exercise ex is a candidate for exercise-slot s of day d AND
    exercise ex2 is a candidate for exercise-slot s of day d AND
    ex1 targets sub muscle group g1 with priority p1 AND
    ex2 targets sub muscle group g2 with priority p2 AND 
    p1 is higher priority than p2
THEN 
    ex2 is no longer a candidate for s

# Filter exercise for the sub same muscle group
IF exercise ex is a candidate for exercise-slot s of day d AND
    exercise ex2 is a candidate for exercise-slot s of day d AND
    ex1 targets sub muscle group g AND
    ex2 targets sub muscle group g
THEN 
    ex2 is no longer a candidate for s

# Prioritizing equipments 
IF exercise ex is a candidate for exercise-slot s of day d AND
    exercise ex2 is a candidate for exercise-slot s of day d AND
    ex1 has equipment type e1 AND
    ex2 has equipment type e2 AND
    user prefers equipment type e1
THEN 
    ex2 is no longer a candidate for s

# Prioritizing  compount movement
IF exercise ex is a candidate for exercise-slot s of day d AND
    exercise ex2 is a candidate for exercise-slot s of day d AND
    ex1 is an compound movement AND
    ex2 is an isolation movement
THEN 
    ex2 is no longer a candidate for s

# Randomly pick one if there are more than 2 options at the end of filtering
IF exercise ex is a candidate for exercise-slot s of day d AND
    exercise ex2 is a candidate for exercise-slot s of day d AND
    ex1 and ex2 has the same priority
    ex1 and ex2 has the same movement type
    ex1 and ex2 are both compound or isolation exercises
THEN 
    ex2 is no longer a candidate for s
