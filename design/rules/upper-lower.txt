


# Day exercise slot (? to be named better) template

A day exercise slot has a target muscle group
A day exercise slot has an order

# Determining the split

IF frequency is 3 days THEN
    IF user preference is Full body THEN
        Workout split is Full body OR
    ELSE
        Workout split is Push-pull-leg
ELSE IF frequency is 4 days OR 5 days THEN
    Workout split is Upper-Lower
ELSE IF frequency is 6 days THEN
    Workout split is Push-pull-leg

# Intializing the days

## Day template

A day has a name
A day has a focus
A day has a list of exercise slots

An exercise slot has a targeted primary muscle group.
An exercise slot has an order (from 1 to n).

## Rules 

IF Workout split is Upper-Lower THEN
    Routine has a day named Upper1
    Upper1 focus is upper body
    Routine has a day named Lower1
    Lower1 focus is lower body
    Routine has a day named Upper2
    Upper2 focus is upper body
    Routine has a day named Lower2
    Lower2 focus is lower body

IF workout split is upper-lower AND frequency is 5 days THEN
    Routine has a day named Shoulder
    Shoulder focus is shoulder

# Initialize template for each day 

IF frequency is 4 days THEN
    Upper days has 7 exercises (e.g Initialize 7 empty exercise slots with order 1...7)
    Lower days has 4 exercises
ELSE 
    Upper day has 6 exercises
    Lower days has 4 exercises

IF Day focus is Upper THEN
    Day requires 2 chest exercises
    Day requires 2 back exercises
    Day requires 1 biceps exercise
    Day requires 1 tricep exercise
    IF frequency is 4 days THEN
        Day requires 1 shoulder exercise

IF exercise slot is 1 THEN
    IF day is Upper THEN
        IF User muscle group focus is chest THEN
            The target muscle group of the exercise slot is chest
        ELSE IF User muscle group focus is back THEN
            The target muscle group of the exercise slot is back
        ELSE
            IF day is Upper1 THEN
                The target muscle group of the exercise slot is chest
            ELSE 
                The target muscle group of the exercise slot is back
    ELSE IF day is Lower THEN
        IF User muscle group focus is quads THEN
            The target muscle group of the exercise slot is quads
        ELSE IF User muscle group focus is hamstring THEN
            The target muscle group of the exercise slot is hamstring
        ELSE IF User muscle group focus is glutes THEN
            The target muscle group of the exercise slot is glutes
        ELSE
            IF day is Lower1 THEN
                The target muscle group of the exercise slot is quads
            THEN
                The target muscle group of the exercise slot is hamstring

FOR ALL day
    FOR ALL exercise slot starting from slot 2
        FOR ALL remaining required muscle groups
            IF the muscle group is not the same as the previous slot AND
                the muscle group has the highest priority among the remaining required muscle groups THEN
                The muscle group is assigned to the slot
    
# Exercise matching

FOR EACH day
    FOR EACH exercise slot of the day
        FOR ALL exercises in fact-base
            IF exercise matches the slot muscle group AND
                exercise is unassigned AND
                the muscle sub-group exercise in fact base has the highest priority among the unassigned exercise of the same primary muscle group AND
                exercise matches the user preferences (free weight or machine)
            THEN 
                exercise in fact base is assigned to the day

# Goal map to reps: https://pmc.ncbi.nlm.nih.gov/articles/PMC7927075/

IF goal is Strength
    Repetition range is 1-5
ELSE IF goal is Hypertrophy
    Repetition range is 8-12
ELSE IF goal is Endurance
    Repetition range is 15+
