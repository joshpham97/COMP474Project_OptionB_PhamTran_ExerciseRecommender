    there is no day
THEN
    Create a day with order 1, name "Upper 1", focus of upper
    Create a day with order 2, name "Lower 1", focus of lower
    Create a day with order 3, name "Upper 2", focus of upper
    Create a day with order 4, name "Lower 2", focus of lower

THEN
    Create an exercise slot with same day order, exercise order of 1, unassigned primary muscle group
    Create an exercise slot with same day order, exercise order of 2, unassigned primary muscle group
    Create an exercise slot with same day order, exercise order of 3, unassigned primary muscle group
    Create an exercise slot with same day order, exercise order of 4, unassigned primary muscle group
    Create an exercise slot with same day order, exercise order of 5, unassigned primary muscle group
    Create an exercise slot with same day order, exercise order of 6, primary muscle group of triceps
    Create an exercise slot with same day order, exercise order of 7, primary muscle group of biceps

THEN
    Create an exercise slot with same day order, exercise order of 1, unassigned primary muscle group
    Create an exercise slot with same day order, exercise order of 2, unassigned primary muscle group
    Create an exercise slot with same day order, exercise order of 3, unassigned primary muscle group
    Create an exercise slot with same day order, exercise order of 4, unassigned primary muscle group

    Create an exercise slot with same day order, exercise order of 6, primary muscle group of glutes
    Create an exercise slot with same day order, exercise order of 7, primary muscle group of calves

; Assign first exercise when user prefers chest or back
IF there is an exercise slot s1 with day order order1 and exercise order 1 and unassigned primary muscle group AND
    there is an exercise slot s2 with day order order2 and exercise order 5 and unassigned primary muscle group AND
    order1 and order2 is the same AND
    the user's preferred muscle group mg is chest or back AND
    mg has same region as the focus of the day with order order1
THEN
    Assign mg as the primary muscle group of s1 AND
    Assign shoulder as the primary muscle group of s2

; Assign first exercise when user prefers shoulder
IF there is an exercise slot s1 with day order order1 and exercise order 1 and unassigned primary muscle group AND
    there is an exercise slot s2 with day order order2 and exercise order 5 and unassigned primary muscle group AND
    order1 and order2 is the same AND
    the user's preferred muscle group is shoulder AND
    shoulder has same region as the focus of the day with order order1
THEN
    Assign shoulder as the primary muscle group of s1 AND
    Assign back as the primary muscle group of s2

; Assign first exercise when user prefers quads
IF there is an exercise slot s with day order of order, exercise order of 1, unassigned primary muscle group AND
    there is a day with same order with a focus f AND
    there is a user input with muscle group of quads AND
    quads has same focus of f
THEN
    Assign quads as the primary muscle group of s

; Assign first exercise when user prefers hamstring
IF there is an exercise slot s with day order of order, exercise order of 1, unassigned primary muscle group AND
    there is a day with same order with a focus f AND
    there is a user input with muscle group of hamstring AND
    hamstring has same focus of f
THEN
    Assign hamstring as the primary muscle group of s

; Assign first exercise when user preference does not match the focus of the day
IF there is an exercise slot s1 with day order order1 and exercise order 1 and unassigned primary muscle group AND
    there is an exercise slot s2 with day order order2 and exercise order 1 and unassigned primary muscle group AND
    the day with order order1 is an upper day AND
    the day with order order2 is an upper day AND
    order1 is less than order2 AND
    the user's preferred muscle group does not belong to the upper region
THEN
    Assign chest as the primary muscle group of s1 AND
    Assign back as the primary muscle group of s2

IF there is an exercise slot s1 with day order order1 and exercise order 1 and unassigned primary muscle group AND
    there is an exercise slot s2 with day order order2 and exercise order 1 and unassigned primary muscle group AND
    the day with order order1 is a lower day AND
    the day with order order2 is a lower day AND
    order1 is less than order2 AND
    the user's preferred muscle group does not belong to the lower region
THEN
    Assign quads as the primary muscle group of s1 AND
    Assign hamstring as the primary muscle group of s2

; Assign alternating muscle groups
IF there is an exercise slot s1 with exercise order ex order 1, primary muscle group chest, day order do1 AND
    there is an exercise slot s2 with exercise order ex order 2, unassigned primary muscle group, day order do2 AND
    do1 and do2 is the same AND
    ex order 2 minus ex order 1 is 1
THEN
    Modify s2's primary muscle group to be back

IF there is an exercise slot s1 with exercise order ex order 1, primary muscle group back, day order do1 AND
    there is an exercise slot s2 with exercise order ex order 2, unassigned primary muscle group, day order do2 AND
    do1 and do2 is the same AND
    ex order 2 minus ex order 1 is 1
THEN
    Modify s2's primary muscle group to be chest

THEN
