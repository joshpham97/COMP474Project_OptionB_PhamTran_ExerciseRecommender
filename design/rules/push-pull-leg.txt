IF Workout split is Push-Pull-Leg THEN
    Workout split has a day named Push1
    Push1 focus is push
    Workout split has a day named Pull1
    Pull1 focus is pull
    Workout split has a day named Leg1
    Leg1 focus is pull
    Workout split has a day named Push2
    Push2 focus is push
    Workout split has a day named Pull2
    Pull2 focus is pull
    Workout split has a day named Leg2
    Leg2 focus is pull

# Initialize template for each day 

IF Day focus is Push THEN
    Day requires 2 chest exercises
    Day requires 2 shoulder exercise
    Day requires 1 triceps exercise
ELSE IF Day focus is Pull THEN
    Day requires 3 back exercises
    Day requires 1 trap exercise
    Day requires 1 triceps exercise
ELSE IF focus is Leg THEN
    Day requires 2 quads exercises
    Day requires 2 hamstring exercises
    Day requires 1 glutes exercises
    Day requires 1 calves exercises

# Picking the first exercise for each day

IF workout split is push-pull-leg AND
    There are 2 different push days AND
    There is no exercise of the day1 assigned an order 1 AND
    There is no exercise of the day2 assigned an order 1 AND
    The exercise s1 of day1 is not assigned an order AND 
    The exercise s1 is a chest exercise AND
    The exercise s2 of day2 is not assigned an order AND
    The exercise s2 is a shoulder exercise
THEN
    The exercise s1 is assigned order 1
    The exercise s2 is assigned order 1

IF workout split is push-pull-leg AND
    There are 2 different pull days AND
    There is no exercise of the day1 assigned an order 1 AND
    There is no exercise of the day2 assigned an order 1 AND
    The exercise s1 of day1 is not assigned an order AND 
    The exercise s1 is a back exercise AND
    The exercise s2 of day2 is not assigned an order AND
    The exercise s2 is a back exercise
THEN
    The exercise s1 is assigned order 1
    The exercise s2 is assigned order 1

IF workout split is push-pull-leg AND
    There are 2 different leg days AND
    There is no exercise of the day1 assigned an order 1 AND
    There is no exercise of the day2 assigned an order 1 AND
    The exercise s1 of day1 is not assigned an order AND 
    The exercise s1 is a quads exercise AND
    The exercise s2 of day2 is not assigned an order AND
    The exercise s2 is a hamstring exercise
THEN
    The exercise s1 is assigned order 1
    The exercise s2 is assigned order 1

# For pull day, we have a special ordering

IF a day focus is pull AND
    An exercise slot s-highest has the highest order AND
    An exercise slot s-unassigned is not assigned AND
    s-highest is a back exercise AND
    s-unassigned is a back exercise
THEN
    The next order is the order of s-highest plus 1
    The next order is assigned to s-unassigned
