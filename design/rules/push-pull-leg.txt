IF Workout split is Push-Pull-Leg THEN
    Workout split has a day named Push1
    Push1 focus is push
    Workout split has a day named Pull1
    Pull1 focus is pull
    Workout split has a day named Leg1
    Leg1 focus is pull
    IF frequency is 6 days
        Workout split has a day named Push2
        Push2 focus is push
        Workout split has a day named Pull2
        Pull2 focus is pull
        Workout split has a day named Leg2
        Leg2 focus is pull

# Initialize template for each day 

IF Day focus is Push THEN
    Day requires 2 chest exercises
    Day requires 2 shoulder exercise
    Day requires 1 triceps exercise
ELSE IF Day focus is Pull THEN
    Day requires 3 back exercises
    Day requires 1 trap exercise
    Day requires 1 triceps exercise
ELSE IF focus is Leg THEN
    Day requires 2 quads exercises
    Day requires 2 hamstring exercises
    Day requires 1 glutes exercises
    Day requires 1 calves exercises

# Picking the first exercise for each day

IF Workout split is Push-Pull-Leg
THEN
    FOR ALL exercise slots AND days AND slot is not assigned a muscle group
        IF exercise slot is 1 AND the exercise slot belongs to the day AND slot is not assigned a muscle group
            IF day is Push1
            THEN The target muscle group of the exercise slot is chest
            ELSE IF day is Push2 
            THEN The target muscle group of the exercise slot is shoulder
            ELSE IF day is Pull1 OR Pull2 
            THEN The target muscle group of the exercise slot is back
            ELSE IF day is Leg1
            THEN The target muscle group of the exercise slot is quads
            ELSE IF day is Leg2
            THEN The target muscle group of the exercise slot is hamstring
