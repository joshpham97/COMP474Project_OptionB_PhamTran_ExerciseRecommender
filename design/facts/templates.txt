# Activity level template

Activity-level is a fuzzy template
Activity-level has the unit of minutes per week
Activity-level has a minimum limit of 0
Activity-level has a maximum limit of 10000
Activity-level has a fuzzy set called sedentary
sedentary is a left-sided right trapezoidal membership function with flat top from 0 to 40, and a slope down from 40 to 60
Activity-level has a fuzzy set called moderate
moderate is a triangle membership function with a peak at 180 and a width of 120
Activity-level has a fuzzy set called active
sedentary is a right-sided right trapezoidal membership function with slope up from 300 to 600 and a flat top from 600 to 10000
